# Processor DPA Checklist

Last updated: 2026-03-30

## Required processors

- Supabase
- OpenRouter
- Resend
- MCP server / token issuer operator

## Checklist

- [ ] Signed DPA in place
- [ ] Subprocessor list reviewed
- [ ] International transfer safeguards documented
- [ ] Data retention/deletion terms verified
- [ ] Security measures reviewed
- [ ] Breach notification clauses confirmed

## Internal record

Store signed agreements and review dates in your internal compliance folder and revisit annually.
