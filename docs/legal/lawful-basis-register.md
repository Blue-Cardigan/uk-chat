# ChatGB Lawful Basis Register

Last updated: 2026-03-30

| Processing activity | Data types | Lawful basis | Notes |
|---|---|---|---|
| Magic-link authentication | Email, auth metadata | Article 6(1)(b) contract | Required to provide access |
| Conversation storage | User prompts, assistant outputs, attachments | Article 6(1)(f) legitimate interests | Needed for continuity and feature operation |
| Council mode orchestration | User issue/scope text, deliberation artifacts | Article 6(1)(f) legitimate interests | Needed to deliver council feature |
| Model usage limits | User ID, model ID, daily counters | Article 6(1)(f) legitimate interests | Abuse prevention and service reliability |
| Public conversation sharing | Shared message content via token | Article 6(1)(a) consent (user action) | Explicit user-triggered share |
| Admin onboarding/invites | Email allowlist, token provisioning state | Article 6(1)(f) legitimate interests | Access management and security |
| Synthetic profile feature (planned) | Demographic/profile inputs | Article 6(1)(a) consent; Article 9(2)(a) explicit consent if special category | Must be optional and revocable |
