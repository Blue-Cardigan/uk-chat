# ChatGB Annual Compliance Review Checklist

Last updated: 2026-03-30
Cadence: annually (and after major feature launches)

## Governance

- [ ] Revalidate lawful basis register
- [ ] Review and refresh privacy notice version
- [ ] Re-run DPIA for new processing activities

## Processor management

- [ ] Confirm DPAs remain current
- [ ] Review processor/subprocessor changes
- [ ] Confirm transfer safeguards for non-UK processing

## Technical controls

- [ ] Validate RLS policies on all personal-data tables
- [ ] Verify retention cron is running and deleting stale records
- [ ] Review logs for PII minimization and access controls
- [ ] Verify account deletion and export paths still function

## User rights operations

- [ ] Test export and deletion workflows end-to-end
- [ ] Review SAR handling response times and evidence

## Audit trail

- [ ] Review admin audit logs for unusual activity
- [ ] Confirm incident/breach runbook ownership and on-call contacts
